## 📖 如何使用本系列

- **推荐入口**：从 **[0卷《每日十二问》](./00_DiscoverySeries_通用入口_v2026-05.pdf)** 开始，无文化背景，人人可用。  
  若希望了解全卷脉络，可接着阅读 **[18卷《发现大学·修身地图》](./18_DiscoverySeries_发现大学_v2026-05.pdf)**，它会用《大学》八目串联所有卷目。  
  若希望深入文化根脉，可阅读 **[17卷《发现传承》](./17_DiscoverySeries_发现传承_v2026-05.pdf)**。  
  在日常工作生活中遇到具体困境（如算法偏见、隐私弹窗、情绪剥削等），可查阅 **[19卷《发现四维·数字时代的底线》](./19_DiscoverySeries_发现四维_v2026-05.pdf)**。18卷是纵向的修身路径，19卷是横向的场景提醒，两者经纬交织，构成一张完整的“数字时代修身网”。
- **进阶选择**：根据您当下的状态（疲惫、迷茫、冲突、育儿焦虑等），直接选对应卷号。
- **学习方式**：每卷独立，无需顺序阅读。每天睡前只问一条，一分钟就好。断了就回来。

所有文档均以 PDF 格式存放在本仓库。
